# @sovrasdk/contracts

Shared zod schemas, types, and the canonical `SovraError` model for the
[Sovra](https://github.com/wsprfme/sovra) extension SDK.

```bash
npm install @sovrasdk/contracts
```

Used by `@sovrasdk/extension-api` and other Sovra packages as the single source of truth for
validation and error codes.
